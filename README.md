# Projeto: Aplicação do Método Iterativo Intercensal no R

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19867085.svg)](https://doi.org/10.5281/zenodo.19867085)

Este projeto teve como objetivo implementar Método Iterativo Intercensal em ambiente R, contribuindo para uma maior visibilida do método e ampliando as possibilidades de aplicação da técnica para construção de tábuas de vida por cor/raça, especialmente em contextos onde a reclassificação racial entre censos demográficos impõe desafios analíticos significativos à pesquisa demográfica.

O projeto reproduz os resultados publicado por Muniz(2023) e faz uma aplicação para estimar as tábuas de vida femininas por raça/cor no Brasil para o período 2012 a 2022.

Como citar:

Gonzaga, M. R. (2026). Expectativa de vida por raça/cor no Brasil entre 2012 e 2022: aplicação do Método Iterativo Intercensal em ambiente R (v1.0.0). Zenodo. https://doi.org/10.5281/zenodo.19867085
